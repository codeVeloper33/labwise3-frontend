Available Experiments

Three core WAEC/NECO physics practicals, fully simulated.

🔵

Simple Pendulum

Vary the pendulum length across multiple trials. Time oscillations with a virtual stopwatch that auto-counts. Plot T² vs L, find g.

g = 4π²L / T²
Free Tier Graph: T² vs L
🟢

Hooke's Law

Add masses to a spring and watch it stretch in real time. Read the new length, compute extension, plot F vs x, find k.

k = F / x

Principle of Moments

Balance a virtual metre rule with weights on both sides. The beam tilts live to show imbalance. Plot M₂ vs M₁.
